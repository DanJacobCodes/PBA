About the Website

This website was built using Angular 2 and follows the Materialize design library for styling. Instead of purchasing a template, I hoped to use this website to reflect my love for simple and effective design, undertake the challenge of building and maintaining something from scratch in order to continue and develop my skills.

Each page is actually a component, along with the footer, and navigation is done through an Angular router, very customary stuff.
